Möchten Sie eine Formel zitieren, können Sie auf die beiden Anführungszeichen klicken und Sie erhalten einen Zitationsvorschlag, den Sie kopieren können. Möchten Sie dagegen die Formel als PDF herunterladen, können Sie das über dieses kleine Symbol hier tun. Dann können Sie einfach darauf klicken, die Datei wird auf Ihrem PC gespeichert und Sie können sie als PDF verwenden. Der jeweilige Zitationsvorschlag findet sich noch mal unten auf jeder einzelnen Seite.
