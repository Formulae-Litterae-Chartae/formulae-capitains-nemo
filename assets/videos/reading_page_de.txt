Dies ist die Leseansicht der Werkstatt. Hier können Sie Dokumente im Ganzen betrachten. Sie sehen, es gibt verschiedene Fußnoten. Um die näher zu betrachten, können Sie einfach auf die jeweilige Fußnote klicken, zum Beispiel auf "g", und dann wird hier am Rand die jeweilige Fußnote angezeigt. Sie sehen außerdem, dass verschiedene Begriffe unterkringelt sind. Das bedeutet, dass für diese Begriffe ein E-Lexikonbeitrag existiert, zum Beispiel für "regnum". Sie können darauf klicken und dann werden Sie zum E-Lexikonbeitrag zu "regnum" geleitet ohne die ursprüngliche Seite zu verlassen. Auch im E-Lexikonbeitrag sehen Sie, dass einige Begriffe unterkringelt sind, zum Beispiel "populus". Wenn Sie diesen Begriff anklicken, werden Sie zum Begriff "populus" geleitet wieder ohne die Seite zu verlassen. Möchten Sie nun den E-Lexikonbeitrag parallel zum Originaltext lesen, dann können Sie auf "Diesen Beitrag parallel lesen" klicken. Der wird dann im Hauptfenster angezeigt. Möchten Sie einen Text schließen, drucken Sie einfach hier oben rechts auf das kleine "X" und der Text verschwindet.
