{{ _('Wortsuche') }}

{{ _("Die Wortsuche durchsucht nur die lateinischen Texte. Wildcard-Zeichen sind erlaubt. Ein Fragezeichen ('?') ersetzt ein einzelnes Zeichen, ein Stern ('*') ein oder mehr Zeichen. Beide funktionieren nur innnerhalb eines Wortes. Bei der Suchanfrage christ* nom* wird jede Reihe von zwei aufeinander folgenden Worten gefunden, bei der das erste Wort mit christ und das zweite mit nom beginnt. Möchten sie im Suchergebnis Worte zwischen beiden Suchbegriffen zulassen, stellen Sie bitte einen entsprechenden 'Suchradius' größer als 0 ein.") }}

{{ _('Vorschlagssuche') }}

{{ _("Die Vorschläge, die in der Wortsuche gezeigt werden, basieren auf alle andere Eingaben in der Suchmaske. Wenn man zum Beispiel Andecavensis als zu durchsuchendes Corpus auswählt, werden nur Vorschläge aus den Formeln von Angers gezeigt. Die oben erwähnten Suchbeschränkungen gelten auch hier. Die Lemmasuche liefert keine Vorschläge aus den Urkunden. Der einzige Parameter, der keinen Einfluss auf die angezeigten Vorschlägen hat, ist 'Wortreihenfolge'. Nur Vorschläge, die die selbe Wortreihenfolge haben, werden angezeigt.") }}

{{ _('Wählt man eine vorgeschlagene Phrase noch bevor man andere Suchparametern einstellt, so haben diese Parameter keinen Einfluss mehr auf die Vorschläge. Das kann dazu führen, dass keine Treffer gefunden werden, weil die gesuchte Phrase in keinem Dokument erscheint, das mit den anderen Suchparametern übereinstimmt. Von daher ist es ratsam, Suchparameter einzustellen, bevor man einen der Vorschläge auswählt.') }}

{{ _('Lemma') }}

{{ _("Mit der Lemmasuche sucht man nach Wörterbuchformen statt flektierten Formen. Eine Lemmasuche nach dominus beispielsweise ruft alle flektierten Formen dieses Wortes wie domini oder Domino. Auch orthographische Varianten wie domni oder Domeno werden durchsucht. Die Lemmasuche funktioniert nur mit den Formeln. Die Lemmatisierungsregeln sind hier zu finden.") }}

{{ _('Wortreihenfolge') }}

{{ _("Ist die Reihenfolge der Worte wichtig, hier klicken. Dadurch wird z.B. bei einer Suche nach uenerabile uir kein Dokument mit uir uenerabile als Treffer angezeigt. Wortreihenfolge wird auch bei einem Suchradius über 0 beachtet. D.h. bei einer Suche nach magnificis uiris mit einem Suchradius von 10 Worten wird magnificis rei puplici uiris angezeit, uiris uenerabilibus adque magnificis aber nicht.") }}

{{ _('Unschärfegrad') }}

{{ _("Der 'Unschärfegrad' legt fest, wie stark ein Ergebnis von der Sucheingabe abweichen darf und trotzdem als gültiges Ergebnis gilt. Die Zahlen quantifizieren, wie viele Abweichungen im Suchergebnis toleriert werden. Unschärfegrad 2 bedeutet z.B., dass zwei Buchstaben im Wort ersetzt, gelöscht, hinzugefügt oder umgesetzt sein dürfen. Wenn man z.B. nach regnum mit einem 'Unschärfegrad' von 2 sucht, werden die Wörter regno und regni aber auch magnum ('m' ersetzt 'r' und 'a' ersetzt 'e') und legum ('l' ersetzt 'r' und 'n' wird gelöscht) gefunden.") }}

{{ _("Mit der Einstellung 'AUTO' wird die erlaubte Zahl solcher Änderungen von der Wortlänge abhängig: bei einem Wort mit 1 oder 2 Buchstaben sind keine Änderungen erlaubt, bei 3-5 Buchstaben ist eine Änderung erlaubt und bei längeren Wörtern sind 2 Änderungen erlaubt. 'AUTO' eignet sich für die meisten Suchanfragen.") }}

{{ _('Suchradius') }}

{{ _("Die maximale Anzahl unzutreffenden Wörter, die zwischen den Suchbegriffen auftreten können. Bei der Suche z.B. nach uenerabilibus uiris mit einem Suchradius von 4 wird uenerabilibus atquae magnificis rei puplici uiris als treffer angezeigt.") }}

{{ _('Datumssuche') }}

{{ _("Da Formeln keine exakte Datumsangabe besitzen, werden bei einer 'Datumssuche' keine Formeln angezeigt.") }}

{{ _('Nach einem bestimmten Datum suchen') }}

{{ _("Liefert alle Suchergebnisse, die den Datumsangaben entsprechen. Gibt man z.B. das Jahr 800 ein, werden Dokumente mit Datierungen im Jahr 800 (z.B. '17. Juni 800') aber auch mit Datumsspannen, die mit dem Jahr 800 überlappen (z.B. '769-813'), als Treffer angezeigt.") }}

{{ _('Datum Plus-Minus') }}

{{ _("Hier wird die erlaubte Abweichung von der eingegebenen Jahresangabe bei der Suche nach einem bestimmten Datum gewählt. Gibt man z.B. '800 Juni 17' mit einem Datum Plus-Minus von 49 Jahren, findet man Dokumente die am 17. Juni zwischen den Jahren 751 und 849 ausgestellt wurden.") }}

{{ _('Nach einer Datumsspanne suchen') }}

{{ _("Bei der Suche nach einer Datumsspanne werden alle Dokumente, die zwischen den Anfangs- und Enddaten liegen als Treffer angezeigt. Wenn man von '750 Juni 17' bis '850 Juni 17' sucht, findet man bespielsweise Dokumente vom 31. Dezember 777 oder 12. Februar 813.") }}

{{ _('Exklusiv') }}

{{ _("Bei einer 'exklusive' Datumsspannesuche sucht man nach eine bestimmte Monats- oder Tagesspanne innerhalb von mehreren Jahren. Sucht man beispielsweise 'exklusiv' vom 17. Juni 800 bis zum 22. November 813, findet man nur Dokumente, die zwischen dem 17. Juni und dem 22. November zwischen den Jahren 800 und 813 ausgestellt wurden. Ein Dokument, das auf den 18. November 803 datiert ist wird als Treffer angezeigt, ein Dokument, das auf den 23. November 803 datiert ist, nicht.") }}
